#CodePenOttawa #2

Grunt Builds and Organizing front end teams.

1. Nathan Ferguson toured us through the custom solution Shopify uses to scale learning HTML and CSS for rapidly growing teams. They use CodePen to test and grade the quality of components and use it to teach over 200 people HTML and CSS in one day.

Base styles for exercises by Nathan Ferguson (@NathanPJF)

2. Tetsuro Takara taught us how to get started with Grunt. We learned how to setup Node, NPM, and how to create our own Gruntfile and package.json.

Now when working our Sass and JavaScript files are automatically combined and minified reducing HTTP requests, our images are compressed, and we no longer have to reload to see our changes. Thanks Tets!
